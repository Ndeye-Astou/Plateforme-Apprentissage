CSS

Maîtrisez l'art de la mise en page et du design web gratuitement. Créez des interfaces élégantes et responsives sans aucun coût.

Cours gratuit Exercices pratiques

Exemple CSS

/* Styles pour le corps de la page */
body {
  background-color: #d0e4fe;
  font-family: 'Segoe UI', sans-serif;
}

/* Styles pour les titres */
h1 {
  color: #0c5e11;
  text-align: center;
  margin-bottom: 1.5rem;
}

/* Styles pour les cartes */
.card {
  border-radius: 15px;
  box-shadow: 0 5px 15px rgba(0,0,0,0.1);
  transition: transform 0.3s ease;
}
.card:hover {
  transform: translateY(-10px);
}

JavaScript

Donnez vie à vos pages web gratuitement. Apprenez à créer des interactions dynamiques et des applications web complexes. Explorez ES6, les promesses et async/await.

Cours gratuit Projets pratiques

Exemple JavaScript

// Sélectionner un élément par son id
const demoElement = document.getElementById('demo');

// Fonction pour changer le style
function changeStyle() {
  demoElement.style.fontSize = "25px";
  demoElement.style.color = "red";
  demoElement.innerHTML = "Texte modifié!";
}

// Ajouter un écouteur d'événement
const myButton = document.getElementById('myBtn');
myButton.addEventListener('click', changeStyle);

// Fonction asynchrone moderne
const fetchData = async () => {
  try {
    const response = await fetch('https://api.example.com/data');
    const data = await response.json();
    console.log(data);
  } catch (error) {
    console.error('Erreur:', error);
  }
};
