Introduction au CSS Débutant

Le CSS (Cascading Style Sheets) est un langage de style utilisé pour définir la présentation d'un document HTML. Il permet de contrôler la mise en forme, le positionnement et l'apparence visuelle des éléments sur une page web.

Pourquoi Utiliser le CSS ?

  • Séparation des Responsabilités : Le CSS permet de séparer la structure (HTML) du style (CSS), facilitant la maintenance et la mise à jour du code.
  • Consistance : Appliquer un ensemble cohérent de styles à travers un site, assurant une apparence uniforme.
  • Adaptabilité : Permet d'ajuster l'apparence d'un site pour différents types d'appareils (responsive design).
Exemple CSS 
/* Style de base pour le corps de la page */
body {
  font-family: Arial, sans-serif;
  background-color: #f4f4f4;
  color: #333;
  line-height: 1.6;
}

/* Style pour les titres */
h1 {
  color: #0c5e11;
  border-bottom: 2px solid #13cd13;
  padding-bottom: 10px;
}

Syntaxe de Base CSS Fondamental

La syntaxe CSS est composée de sélecteurs et de déclarations. Un sélecteur pointe vers l'élément HTML que vous souhaitez styliser. Le bloc de déclaration contient une ou plusieurs déclarations séparées par des points-virgules.

Structure CSS 
sélecteur {
  propriété: valeur;
  autre-propriété: autre-valeur;
}

Types de Sélecteurs Essentiel

Les sélecteurs CSS sont utilisés pour "trouver" (ou sélectionner) les éléments HTML que vous souhaitez styliser.

  • Sélecteur d'Élément HTML : Sélectionne tous les éléments du type spécifié.
  • Sélecteur de Classe : Sélectionne tous les éléments avec la classe spécifiée.
  • Sélecteur d'ID : Sélectionne un élément avec l'ID spécifié.
Exemples de sélecteurs 
/* Sélecteur d'élément */
p {
  font-size: 16px;
}

/* Sélecteur de classe */
.important {
  font-weight: bold;
  color: #ff6b35;
}

/* Sélecteur d'ID */
#header {
  background-color: #1a1a1a;
  color: white;
  padding: 20px;
}

SASS et SCSS Avancé

SASS (Syntactically Awesome Stylesheets) et SCSS (Sassy CSS) sont des préprocesseurs CSS qui ajoutent des fonctionnalités supplémentaires comme les variables, les mixins, et les imports.

Exemple SASS 
$couleurPrincipale: #3498db
body
  background-color: $couleurPrincipale
  font-family: 'Arial', sans-serif

.container
  width: 80%
  margin: 0 auto
Exemple SCSS 
$couleurPrincipale: #3498db;
$couleurSecondaire: #ff6b35;

body {
  background-color: $couleurPrincipale;
  font-family: 'Arial', sans-serif;
}

.container {
  width: 80%;
  margin: 0 auto;
  
  .title {
    color: $couleurSecondaire;
    font-size: 2rem;
  }
}

Méthodes d'Intégration du CSS Fondamental

Il existe trois principales méthodes pour intégrer du CSS dans une page HTML :

  • Balise style : Utiliser la balise <style> dans l'en-tête HTML
  • Attribut style : Utiliser l'attribut style directement dans les balises HTML
  • Fichier Externe : Créer un fichier CSS externe et l'associer avec la balise <link>
Exemple Fichier Externe 
<!DOCTYPE html>
<html lang="fr">
<head>
  <link rel="stylesheet" href="style.css">
  <title>Titre de la page</title>
</head>
<body>
  <!-- Contenu de la page -->
</body>
</html>

Propriétés Courantes Essentiel

Voici quelques-unes des propriétés CSS les plus couramment utilisées :

  • color : Définit la couleur du texte.
  • font-size : Définit la taille de la police.
  • font-family : Définit la police de caractères.
  • background-color : Définit la couleur de fond.
  • margin et padding : Contrôlent les marges et les espacements internes.
  • border : Définit la bordure autour d'un élément.
  • display : Définit le type de boîte de rendu d'un élément.

CSS : les alignements Intermediaire

Le Flexbox et le Positionnement en CSS

Le positionnement des éléments en CSS est essentiel pour créer des mises en page flexibles et réactives. Deux techniques importantes sont largement utilisées : le Flexbox et les propriétés de positionnement.

Exemple Flexbox 
.container {
  display: flex;
  justify-content: center; /* Centrage horizontal */
  align-items: center; /* Centrage vertical */
  gap: 20px; /* Espacement entre les éléments */
}

.item {
  flex: 1; /* Les éléments s'étendent pour remplir l'espace */
  padding: 15px;
  background: #f0f0f0;
  border-radius: 8px;
}

CSS : Le Display Grid Intermediaire

Le Display Grid crée une grille bidimensionnelle composée de lignes et de colonnes, dans laquelle les éléments HTML peuvent être placés de manière précise.

Exemple Grid 
.container {
  display: grid;
  grid-template-columns: 1fr 1fr 1fr; /* Trois colonnes égales */
  grid-template-rows: 100px auto 100px;
  grid-gap: 20px; /* Espacement entre les éléments */
}

.header {
  grid-column: 1 / 4;
  background: #0c5e11;
  color: white;
  padding: 20px;
}

.sidebar {
  grid-row: 2 / 4;
  background: #f0f0f0;
  padding: 20px;
}

.main {
  grid-column: 2 / 4;
  background: white;
  padding: 20px;
}

CSS : Les Animations Avancé

Les animations CSS permettent de créer des transitions fluides et des animations complexes sans JavaScript.

Exemple d'animation 
@keyframes bounce {
  0%, 100% { 
    transform: translateY(0); 
  }
  50% { 
    transform: translateY(-20px); 
  }
}

.element {
  animation: bounce 1s infinite;
  width: 100px;
  height: 100px;
  background: #ff6b35;
  border-radius: 50%;
}

CSS : Les Sélecteurs avancés Avancé

Les sélecteurs avancés en CSS offrent une précision accrue pour cibler des éléments spécifiques dans une page web.

Exemples de sélecteurs avancés 
/* Sélectionne le premier enfant */
li:first-child {
  font-weight: bold;
  color: #0c5e11;
}

/* Sélectionne les éléments pairs dans un tableau */
tr:nth-child(even) {
  background-color: #f2f2f2;
}

/* Sélectionne les liens visités */
a:visited {
  color: #800080;
}

/* Sélectionne les inputs de type texte */
input[type="text"] {
  border: 2px solid #4a90e2;
  padding: 8px;
}

/* Sélectionne les paragraphes qui suivent immédiatement un h2 */
h2 + p {
  font-style: italic;
  color: #666;
}
