Bootstrap 5 : comment démarrer

Qu'est-ce que Bootstrap ?

Bootstrap est un framework front-end gratuit pour un développement Web plus rapide et plus facile

Bootstrap comprend des modèles de conception basés sur HTML et CSS pour la typographie, les formulaires, les boutons, les tableaux, la navigation, les modaux, les carrousels d'images et bien d'autres, ainsi que des plugins JavaScript en option

Bootstrap vous donne également la possibilité de créer facilement des designs réactifs

Qu'est-ce que le Responsive Web Design ?

Le Responsive Web Design consiste à créer des sites Web qui s'ajustent automatiquement pour s'afficher correctement sur tous les appareils, des petits téléphones aux grands ordinateurs de bureau.

<h1>Ma premiere Page Bootstrap</h1>
<p>Redimensionnez cette page réactive pour voir l'effet !</p>
</div>
<div class="container mt-5">
<div class="row">
<div class="col-sm-4">

Versions de Bootstrap

Bootstrap 5 (sorti en 2021) est la dernière version de Bootstrap (sortie en 2013) ; avec de nouveaux composants, une feuille de style plus rapide et plus de réactivité.

Bootstrap 5 prend en charge les dernières versions stables de tous les principaux navigateurs et plates-formes. Cependant, Internet Explorer 11 et les versions antérieures ne sont pas pris en charge.

La principale différence entre Bootstrap 5 et Bootstrap 3 et 4 est que Bootstrap 5 est passé à JavaScript vanilla au lieu de jQuery.

Remarque : Bootstrap 3 et Bootstrap 4 sont toujours pris en charge par l'équipe pour les corrections de bogues critiques et les modifications de la documentation, et il est parfaitement sûr de continuer à les utiliser. Cependant, de nouvelles fonctionnalités ne leur seront PAS ajoutées.

Avantages de Bootstrap :
  • Facile à utiliser : toute personne possédant des connaissances de base en HTML et CSS peut commencer à utiliser Bootstrap
  • Fonctionnalités réactives : le CSS réactif de Bootstrap s'adapte aux téléphones, tablettes et ordinateurs de bureau
  • Approche mobile d'abord : dans Bootstrap, les styles mobiles d'abord font partie du framework de base
  • Compatibilité du navigateur : Bootstrap 5 est compatible avec tous les navigateurs modernes (Chrome, Firefox, Edge, Safari et Opera). Notez que si vous avez besoin de la prise en charge d'IE 11 et des versions antérieures, vous devez utiliser BS4 ou BS3.
Où obtenir Bootstrap 5 ?

Il existe deux façons de commencer à utiliser Bootstrap 5 sur votre propre site Web.

Vous pouvez :

  • Inclure Bootstrap 5 à partir d'un CDN
  • Télécharger Bootstrap 5 depuis getbootstrap.com
Bootstrap 5 CDN

Si vous ne souhaitez pas télécharger et héberger Bootstrap 5 vous-même, vous pouvez l'inclure à partir d'un CDN (Content Delivery Network).

jsDelivr fournit un support CDN pour CSS et JavaScript de Bootstrap :

Exemple

<!DOCTYPE html>
<html>
<title>Titres HTML</title>

<!-- Latest compiled and minified CSS -->
<link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/css/bootstrap.min.css" rel="stylesheet">
<!-- Latest compiled JavaScript --> <script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/js/bootstrap.bundle.min.js"></script> </body>
</html>

L'un des avantages de l'utilisation du CDN Bootstrap 5 :

De nombreux utilisateurs ont déjà téléchargé Bootstrap 5 depuis jsDelivr lorsqu'ils visitent un autre site. Par conséquent, il sera chargé à partir du cache lorsqu'ils visiteront votre site, ce qui accélère le temps de chargement. De plus, la plupart des CDN s'assureront qu'une fois qu'un utilisateur demandera un fichier, il sera servi à partir du serveur le plus proche de lui, ce qui accélère également le temps de chargement.

JavaScript ?

Bootstrap 5 utilise JavaScript pour différents composants (comme les modaux, les info-bulles, les popovers, etc.). Cependant, si vous utilisez simplement la partie CSS de Bootstrap, vous n'en avez pas besoin.

Téléchargement de Bootstrap 5

Si vous souhaitez télécharger et héberger Bootstrap 5 vous-même, accédez à getbootstrap.com et suivez les instructions qui y sont données.

Créez votre première page Web avec Bootstrap 5
  1. Ajoutez le doctype HTML5

    2. Bootstrap 5 utilise des éléments HTML et des propriétés CSS qui nécessitent le doctype HTML5.

    Incluez toujours le doctype HTML5 au début de la page, ainsi que l'attribut lang et le titre et le jeu de caractères corrects :

    Exemple HTML

    <!DOCTYPE html>
    <html lang="fr">
    <head>
    <meta charset="UTF-8">
    <title>Exemple Bootstrap 5</title>
    </body>
    </html>
  2. Bootstrap 5 est avant tout destiné aux appareils mobiles

    3. Bootstrap 5 est conçu pour être réactif aux appareils mobiles. Les styles mobiles font partie du framework de base.

    Pour garantir un rendu correct et un zoom tactile, ajoutez la <meta> balise suivante à l'intérieur de l' <head> élément :

    Exemple HTML

    <meta name="viewport" content="width=device-width, initial-scale=1">

    La partie width=device-width définit la largeur de la page pour qu'elle suive la largeur de l'écran de l'appareil (qui varie en fonction de l'appareil).

    La partie initial-scale=1 définit le niveau de zoom initial lorsque la page est chargée pour la première fois par le navigateur.

  3. Conteneurs

    4. Bootstrap 5 nécessite également un élément conteneur pour envelopper le contenu du site.

    Vous avez le choix entre deux classes de conteneurs :

    • La .container classe fournit un conteneur réactif à largeur fixe
    • La .container-fluid classe fournit un conteneur pleine largeur, couvrant toute la largeur de la fenêtre d'affichage
Deux pages de Bootstrap de base de 5

L'exemple suivant montre le code d'une page Bootstrap 5 de base (avec un conteneur réactif à largeur fixe) :

Exemple de conteneur

<!DOCTYPE html>
<html lang="fr">
<head>
<meta charset="UTF-8">
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<title>Exemple Bootstrap</title>
<link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/css/bootstrap.min.css" rel="stylesheet">
<script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/js/bootstrap.bundle.min.js">
</script>
</head>
<body>

<div class="container">
<h1>Ma première page Bootstrap</h1>
<p>Cette partie est à l'intérieur d'une classe container</p>
<p>La classe .container fournit un conteneur à largeur fixe et réactif.</p>
</div>

</body>
</html>
