Les Fonctions en JavaScript

Introduction

- Les fonctions sont des blocs de code réutilisables qui permettent d'organiser et de structurer votre code. Elles prennent des entrées (appelées paramètres), effectuent des opérations et retournent une valeur.

Déclaration de Fonction
  1. Fonction Nommée

    2. Une fonction nommée est définie avec le mot-clé function suivi d'un nom, de paramètres entre parenthèses et d'un bloc de code entre accolades.

    Exemple

    function saluer(nom) {
    return "Bonjour " + nom + "!";
    }
    console.log(saluer("Alice")); // Affiche "Bonjour Alice!"
  2. Fonction Anonyme

    3. Une fonction anonyme est une fonction sans nom, souvent utilisée comme valeur de retour ou argument pour une autre fonction.

    Exemple

    let afficherMessage = function(message) {
    console.log(message);
    };
    afficherMessage("Salut!"); // Affiche "Salut!"
  3. Fonction Fléchée (Arrow Function)

    4. Les fonctions fléchées offrent une syntaxe plus concise et ne lient pas leur propre this.

    Exemple

    let addition = (a, b) => a + b;
    console.log(addition(2, 3)); // Affiche 5
  4. Fonctions Imbriquées

    5. Les fonctions peuvent être définies à l'intérieur d'autres fonctions.

    Exemple

    function exterieur(a) {
    function interieur(b) {
    return a + b;
    }
    return interieur;
    }
    let additionDeCinq = exterieur(5);
    console.log(additionDeCinq(3)); // Affiche 8
Fonctions Prédéfinies (Built-in Functions)

JavaScript inclut de nombreuses fonctions prédéfinies qui peuvent être utilisées immédiatement.

  1. parseInt()

    2. Convertit une chaîne en entier.

    Exemple

    let entier = "123";
    console.log(typeof entier);// Affiche string
    entier = parseInt("123");
    console.log(entier); // Affiche 123
    console.log(typeof entier); // Affiche number
  2. parseFloat()

    3. Convertit une chaîne en nombre à virgule flottante.

    Exemple

    let flottant = parseFloat("123.45");
    console.log(flottant); // Affiche 123.45
  3. isNaN()

    4. Vérifie si une valeur est NaN (Not-a-Number).

    Exemple

    let resultat = isNaN("123");
    console.log(resultat); // Affiche false
  4. setTimeout()

    5. Exécute une fonction après un certain délai.

    Exemple

    setTimeout(function() {
    console.log("Exécuté après 2 secondes");
    }, 2000);
  5. setInterval()

    6. Exécute une fonction à intervalles réguliers.

    Exemple

    let compteur = 0;
    let intervalId = setInterval(function() {
    compteur++;
    console.log("Compteur : " + compteur);
    if (compteur === 5) {
    clearInterval(intervalId);
    }
    }, 1000);
Fonctions Fonctions Avancées
  1. Fonctions de Rappel (Callback Functions)

    2. Les fonctions de rappel sont des fonctions passées en argument à une autre fonction, qui les appelle après avoir terminé son travail.

    Exemple

    function demanderNom(callback) {
    let nom = prompt("Quel est votre nom?");
    callback(nom);
    }
    demanderNom(function(nom) {
    console.log("Bonjour " + nom + "!");
    });
  2. Fonctions Immédiatement Invocables (IIFE)

    3. Les fonctions immédiatement invoquées sont définies et exécutées immédiatement.

    Exemple

    (function() {
    console.log("Cette fonction est immédiatement invoquée.");
    })();
  3. Fonctions Asynchrones

    4. Les fonctions asynchrones utilisent async et await pour gérer les opérations asynchrones de manière plus lisible.

    Exemple

    async function fetchData() {
    let response = await fetch('https://api.example.com/data');
    let data = await response.json();
    console.log(data);
    }
    fetchData();
    Portée des Variables

    Les variables en JavaScript peuvent avoir une portée globale ou locale.

    • Globale : Déclarée en dehors de toute fonction.
    • Locale : Déclarée à l'intérieur d'une fonction.

    Exemple

    let globalVar = "Je suis globale";
    function afficherGlobale() {
    console.log(globalVar);
    }
    afficherGlobale(); // Affiche "Je suis globale"
    Récursivité

    Une fonction récursive est une fonction qui s'appelle elle-même.

    Exemple

    function factorielle(n) {
    if (n <= 1) {
    return 1;
    } else {
    return n * factorielle(n - 1);
    }
    }
    console.log(factorielle(5)); // Affiche 120
    Spread Operator

    Le spread operator (...) permet d'étaler les éléments d'un tableau ou les propriétés d'un objet. Il est particulièrement utile dans les fonctions pour accepter un nombre variable d'arguments ou pour étendre des objets.

    1. Utilisation avec les Tableaux

      2. Exemple

      let nombres = [1, 2, 3];
      let plusDeNombres = [...nombres, 4, 5, 6];
      console.log(plusDeNombres); // Affiche [1, 2, 3, 4, 5, 6]
    2. Utilisation avec les Objets

      3. Exemple

      let personne = { nom: "Alice", age: 25 };
      let personneAvecVille = { ...personne, ville: "Paris" };
      console.log(personneAvecVille); // Affiche { nom: "Alice", age: 25, ville: "Paris"
      }
    3. Utilisation dans les Fonctions

      4. Le spread operator peut être utilisé pour passer un nombre variable d'arguments à une fonction.

      Exemple

      function addition(...nombres) {
      return nombres.reduce((acc, curr) => acc + curr, 0);
      }
      console.log(addition(1, 2, 3, 4)); // Affiche 10
