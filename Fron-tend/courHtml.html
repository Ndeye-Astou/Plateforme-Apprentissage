Maîtriser HTML : Fondations du Web

Qu'est-ce que HTML ?

HTML (HyperText Markup Language) est le langage standard utilisé pour créer et structurer le contenu des pages web. C'est la fondation de tout site web, qui définit la structure et le contenu d'une page.

Analogie pratique

Pensez à HTML comme à la structure d'une maison :

  • Les <h1> et <h2> sont les titres principaux (fondations)
  • Les <p> sont les murs (contenu principal)
  • Les <img> sont les fenêtres (éléments visuels)
  • Les <a> sont les portes (liens entre les pièces)

Sans HTML, un site web serait comme une maison sans structure - impossible à construire !

Comment fonctionne HTML ?

HTML fonctionne avec un système de balises qui entourent le contenu pour indiquer au navigateur comment l'afficher :

Balises d'ouverture et fermeture

La plupart des éléments HTML ont une balise d'ouverture <balise> et une balise de fermeture </balise> avec le contenu entre les deux.

Rôle du navigateur

Le navigateur web (Chrome, Firefox, etc.) lit le code HTML et le transforme en page visuelle que les utilisateurs peuvent voir et interagir avec.

Évolution d'HTML

1991 - HTML 1.0

Première version avec 18 balises seulement

1995 - HTML 2.0

Standardisation par l'IETF

1997 - HTML 4.0

Introduction des CSS et support multimédia

2014 - HTML5

Version moderne avec sémantique améliorée et API avancées

Structure de Base d'une Page HTML

Toute page HTML suit une structure fondamentale qui sert de squelette à votre contenu :

structure.html 
<!DOCTYPE html>
<html lang="fr">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Titre de la page</title>
  <!-- Métadonnées et liens CSS -->
</head>
<body>
  <!-- Contenu visible de la page -->
  <header>
    <!-- En-tête de page -->
  </header>
  
  <main>
    <!-- Contenu principal -->
    <section>
      <h1>Titre principal</h1>
      <p>Paragraphe de texte</p>
    </section>
  </main>
  
  <footer>
    <!-- Pied de page -->
  </footer>
</body>
</html>
<!DOCTYPE html>

Déclaration obligatoire pour indiquer la version HTML (HTML5)

<head>

Contient les métadonnées, liens CSS, et autres informations non visibles

<body>

Contient tout le contenu visible de la page (texte, images, etc.)

Quelle balise contient le contenu principal visible d'une page web ?
<body>
<head>
<html>
<main>
Correct ! La balise <body> contient tout le contenu visible de la page.

Éléments de base en HTML

Paragraphe <p>

La balise <p> définit un paragraphe de texte. C'est l'élément de base pour le contenu textuel.

paragraphe.html 
<p>Ceci est un paragraphe de texte. En HTML, les paragraphes sont des éléments de bloc qui structurent votre contenu.</p>
<p>Un deuxième paragraphe montre comment le navigateur ajoute automatiquement un espace entre les éléments.</p>
Résultat :

Ceci est un paragraphe de texte. En HTML, les paragraphes sont des éléments de bloc qui structurent votre contenu.

Un deuxième paragraphe montre comment le navigateur ajoute automatiquement un espace entre les éléments.

En-têtes <h1> à <h6>

Les balises d'en-tête créent une hiérarchie de titres essentielle pour la structure et le référencement.

titres.html 
<h1>Titre Principal (Niveau 1)</h1>
<h2>Section Importante (Niveau 2)</h2>
<h3>Sous-section (Niveau 3)</h3>
<h4>Titre Niveau 4</h4>
<h5>Titre Niveau 5</h5>
<h6>Titre Niveau 6</h6>
Résultat :

Titre Principal (Niveau 1)

Section Importante (Niveau 2)

Sous-section (Niveau 3)

Titre Niveau 4

Titre Niveau 5
Titre Niveau 6
Bonnes pratiques : Utilisez une seule balise <h1> par page pour votre titre principal. Cela aide les moteurs de recherche à comprendre votre contenu.

Listes en HTML

Liste Non Ordonnée

Pour les éléments sans ordre particulier (liste à puces)

<ul>
  <li>Élément 1</li>
  <li>Élément 2</li>
</ul>
Résultat :
  • Élément 1
  • Élément 2
Liste Ordonnée

Pour les éléments séquentiels (liste numérotée)

<ol>
  <li>Première étape</li>
  <li>Deuxième étape</li>
</ol>
Résultat :
  1. Première étape
  2. Deuxième étape

Tableaux en HTML

Les tableaux organisent les données en lignes et colonnes :

tableau.html 
<table>
  <tr>
    <th>Nom</th>
    <th>Âge</th>
    <th>Ville</th>
  </tr>
  <tr>
    <td>Marie</td>
    <td>28</td>
    <td>Paris</td>
  </tr>
  <tr>
    <td>Pierre</td>
    <td>32</td>
    <td>Lyon</td>
  </tr>
</table>
Résultat :
Nom Âge Ville
Marie 28 Paris
Pierre 32 Lyon
Éléments d'un tableau :
  • <table> : Conteneur principal
  • <tr> : Définit une ligne de tableau
  • <th> : Cellule d'en-tête (gras et centré)
  • <td> : Cellule standard de données

Formulaires en HTML

Les formulaires permettent aux utilisateurs d'interagir avec votre site :

formulaire.html 
<form action="/traitement" method="POST">
  <label for="nom">Nom :</label>
  <input type="text" id="nom" name="nom" required>
  
  <label for="email">Email :</label>
  <input type="email" id="email" name="email" required>
  
  <label for="message">Message :</label>
  <textarea id="message" name="message"></textarea>
  
  <button type="submit">Envoyer</button>
</form>

Conclusion

HTML est la fondation indispensable de tout site web. Bien que simple à apprendre, sa maîtrise est essentielle pour créer des sites bien structurés, accessibles et optimisés pour le référencement.
